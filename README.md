# Shah Textile Portal

A modern textile catalog admin portal built using React JS.

The portal allows textile businesses to manage fabric products, customer enquiries, and catalog listings through a clean and simple admin dashboard.

---

## Features

### Admin Authentication
- Login Page UI
- Dashboard Redirection

### Dashboard
- Product Overview
- Recent Products
- Recent Enquiries

### Product Management
- Add New Products
- Upload Fabric Images
- Edit Products
- Delete Products
- Product Status (Active / Draft)

### Product Information
- Fabric Name
- Fabric Type
- Width
- Composition
- Min Order
- Delivery Time
- Available Colors

### Customer Enquiries
- View Customer Details
- Fabric Requirements
- Quantity Requested
- Color Preference
- Mark Enquiries as Completed

---

## Installation

Clone the repository:

```bash
git clone https://github.com/ayushiptl25/Shah-Textile.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
